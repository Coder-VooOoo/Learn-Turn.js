Turn.js
=========

### 拷贝 *Turn.js v1*

turn.js是jQuery的一个插件，它添加了一个漂亮的过渡，类似于书籍或杂志中的真实页面。它适用于所有现代浏览器，包括触摸设备。

### `turned` 事件的定义和触发

`bind()` 与 `trigger()`

### 自执行函数IIFE

主要用于创建一个局部的作用域，避免变量和函数污染全局命名空间
